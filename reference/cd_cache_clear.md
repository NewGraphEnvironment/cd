# Clear the cd cache

Removes all cached files from the cd cache directory.

## Usage

``` r
cd_cache_clear(cache_dir = NULL)
```

## Arguments

- cache_dir:

  Character. Override the default cache location.

## Value

Invisibly returns the number of files removed.

## Examples

``` r
if (FALSE) { # \dontrun{
cd_cache_clear()
} # }
```
