# Task Plan — WSG x ecoregion overlap (#47)

## Goal

Add a small WSG x ecoregion overlap section to the `peace-fwcp` vignette so readers can map the per-ecoregion climate-departure story onto the watershed-group reporting unit.

## Steps

- [ ] **Compute the WSG x ecoregion area matrix** — for each of the 18 WSGs, the percent of its total area falling in each of the 5 ecoregions. Use `sf::st_intersection` on the bundled `wsgs` and `ecoregions` layers in `inst/extdata/context_fwcp_peace.gpkg`. Cheap, fully local (no S3).
- [ ] **Compute or curate commentary per WSG** — a one-line plain-language note. Try a templated version first (e.g. "Entirely within X" if >95%, "Largely X (N%) with Y (M%)" for split cases). If templated text reads poorly for some rows, hand-curate those.
- [ ] **Ship as CSV** — `inst/extdata/peace_wsg_ecoregion_commentary.csv` with columns: `wsg_code`, `wsg_name`, `area_km2`, `OMM_pct`, `BMP_pct`, `CRM_pct`, `FAB_pct`, `NRM_pct`, `commentary`. Generated by a `data-raw/` script.
- [ ] **`data-raw/peace_wsg_ecoregion_commentary.R`** — generator script. Reads bundled gpkg, computes intersection percentages, generates templated commentary (with override hooks for hand-curated rows if needed).
- [ ] **Vignette section** — between Per-Ecoregion Variation and Interpretation (or after Interpretation depending on flow). New section heading "Watershed Groups Across Ecoregions". Reads the CSV, renders as a kable.
- [ ] **Update Interpretation** — add a paragraph tying the precipitation-trending-up signal (BMP, NRM) to the WSGs that sit in those ecoregions, vs the WSGs sitting in FAB / CRM / OMM where precipitation shows no significant change.
- [ ] **Render local** — confirm timing stays fast (no S3 calls added).
- [ ] **Code-check** before commit.
- [ ] **PR** with SRED ref in body.

## Out of scope (per issue body)

- General `cd_overlap()` helper function.
- Per-WSG climate departure (~18 cd_extract calls).

## Validation

- [ ] WSG percentages sum to ~100% per row (all WSG area is accounted for, modulo tiny slivers outside any ecoregion polygon)
- [ ] Total km^2 per WSG is sane
- [ ] Vignette still renders in <30 s
- [ ] CI passes on PR
